
编译工具：
1.   VS 2012
2.   安装 Qt 5.1.0 x86 (32位)
	（下载地址：download.qt.io/archive/qt/）
3.   安装 Visual Studio Add-in 1.2.5 for Qt5 
	（下载地址：download.qt.io/official_releases/vsaddin/qt-vs-addin-1.2.5.exe）
	这样安装好了之后，VS 工具中会出现Qt5插件的菜单,点击Qt Optios 查看Qt Versions中是否有安装的Qt 5.0.1，
	如果没有，则添加，点击Add， 输入Qt 名字， 文件路径为 安装Qt的路径，
	例如：F:\Qt\Qt5.1.0\5.1.0\msvc2012_64_opengl （Qt中Bin文件的上一级目录）
	
Note:
   IFCTools工程点击右键 Qt Project Setting 里面选择工程使用的Qt版本
